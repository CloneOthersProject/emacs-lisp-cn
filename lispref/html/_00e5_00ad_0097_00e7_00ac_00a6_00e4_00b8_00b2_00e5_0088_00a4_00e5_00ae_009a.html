4.2 字符串判定

关于序列以及数组的通用判定的更多信息，请参考Sequences Arrays Vectors.以及Arrays.。

— Function: stringp object

此函数在object是字符串时返回t，否则返回nil。

— Function: string-or-null-p object

此函数在object是字符串或是nil时返回t，否则返回nil。

— Function: char-or-string-p object

此函数在object是字符串或是字符时时返回t，否则返回nil。
