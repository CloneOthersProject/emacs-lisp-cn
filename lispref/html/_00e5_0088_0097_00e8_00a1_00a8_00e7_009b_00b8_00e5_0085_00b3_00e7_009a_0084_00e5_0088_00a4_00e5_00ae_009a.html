5.2 列表相关的判定

以下判定测试一个Lisp对象是否是一个原子，是否是一个cons cell或列表，是否是特定的对象nil。（这里的许多判定可以由其他判定定义，但由于它们使用的很频繁，值得 将它们全部定义出来。）

— Function: consp object

此函数在object是cons cell时返回t，否则返回nil。尽管nil是一个列表 ，但它不是一个cons cell。

— Function: atom object

此函数在object是原子时返回t，否则返回nil。除cons cell外的对象都是原子。符号 nil即是一个原子，又是一个列表，它是唯一同属于两类的Lisp对象。 that is both. 

          (atom object) == (not (consp object))
— Function: listp object

此函数在object是cons cell或nil时返回t，否则返回nil。 

          (listp '(1))
               ⇒ t
          (listp '())
               ⇒ t
— Function: nlistp object

此函数是listp的反面：它在object不是列表时返回t，否则返回nil。 

          (listp object) == (not (nlistp object))
— Function: null object

此函数在object是nil时返回t，否则返回nil。此函数等同于not，但为 了更加清晰，我们在object被当作列表时使用它，而在object被当作真值时使用not。（请参考 Combining Conditions.中的not） 

          (null '(1))
               ⇒ nil
          (null '())
               ⇒ t
